b. Make hormones to control anemia
c. Eliminate excess fluids
1. a+b
2. b only
3. all of the above

2. Low salt intake can lower blood pressure
True
False

3. Peritoneal dialysis requires the creation of an A-V Fistula
True
False

4. There are two types of dialysis, hemodialysis and peritoneal dialysis
True
False

5. Major ways of preventing renal failure may include (choose all that apply)
a. Blood pressure control
b. An angiotensin blocker
c. Dietary protein restriction
1. a+c only
2. b only
3. all of the above

6. Early placement of an A-V fistula can frequently eliminate starting dialysis as a hospital patient
True
False

7. The kidney makes an active form of vitamin D which controls the parathyroid gland
True
False

True
False

9. Acid accumulation may decrease the appetite in early kidney failure
True
False

False
